# 我想听歌！

> 在 AstrBot 聊天里听歌、找歌和下载歌曲。

说出想听的歌，插件会直接播放一首合适的歌曲。想比较版本或下载歌曲时，插件会列出候选，由你确认后再继续。

## 怎么用

- `放首晴天`、`来一首周杰伦的晴天`：直接听歌。
- `唱首歌`：推荐一首合适的歌曲并播放。
- `下载晴天`：显示最多十个候选；回复 `1 下载` 下载。
- `搜索歌曲 晴天`：显示最多十个候选；回复 `1` 听歌、`1 下载` 下载，或回复 `取消` 结束。
- `下载 BV1...`、`下载 av170001` 或 `下载 <普通 Bilibili 视频链接>`：精确打开该视频；多 P 视频会列出各分 P 供你选择。

候选列表只显示视频标题和时长，不显示 UP 主。下载始终需要先确认候选，避免把不是你想要的版本保存下来。

直接听歌支持最长 15 分钟、最大 25 MiB 的音频。超过 15 分钟的候选会标记“仅可下载”；回复“序号 下载”即可发送文件。下载不限制歌曲时长，单个文件最大 100 MiB。获取音频时，语音总共最多等待 3 分钟，文件下载总共最多等待 15 分钟（包括必要的备用线路尝试）；超过文件大小限制时，插件会明确提示无法发送。

想听现场、翻唱、AI 创作、Remix 等特定版本时，请直接写在请求里。候选会保留这些标题标记，方便按你的需求选择。

## 管理员设置

在 AstrBot WebUI 的“我想听歌！”页面登录 Bilibili 账号。只有 Dashboard 管理员可以操作登录；聊天用户不需要登录。未登录时仍可搜索和尝试播放，但部分内容可能受平台限制。

音频只在发送期间临时保存，语音或文件发送完成后都会自动清理。
个人微信（`weixin_oc`）不支持本插件使用的语音消息类型时，会自动收到同一首歌的音频文件。

## 安装与更新

首次安装时，请在 AstrBot WebUI 中通过以下仓库地址添加插件：

`https://github.com/57Darling02/astrbot_plugin_listen_music`

之后可直接在 WebUI 的插件页面检查和安装更新。若此前是手动复制目录或上传 ZIP 安装，请先在 WebUI 中将插件源绑定为该仓库，或重新通过上述地址安装一次；AstrBot 需要保存安装来源后才能在线更新。

## 运行要求

- AstrBot `>=4.26,<5`
- 宿主机已安装 `ffmpeg`，用于发送语音和下载文件
